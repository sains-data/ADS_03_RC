## How to run script
1. Buka file code yang ada di folder code
2. run code (Jika ada package yang belum di install silahkan diinstal dengan install.package() )


## Package 
1. library(tidyverse) -> Visualisasi data 
2. library(readr) -> Pembaca csv
3. library(dplyr) -> data manipulasi 
4. library(stringr) -> manipulasi string
